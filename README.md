# Blackjack Tracker
Blackjack game tracker using OpenCV and Python

## Usage
Clone the repository:
```
git clone https://github.com/martinabeleda/Blackjack-Tracker
```

Run the code using a usb webcam:
```
cd Blackjack-Tracker
python3 main.py
```
